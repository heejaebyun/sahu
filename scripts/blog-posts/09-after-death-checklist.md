# 사망 후 해야할 일 체크리스트 — 시간순 정리 (2026년 최신)

가족이 돌아가시면 슬픔 속에서도 해야 할 행정 절차가 정말 많습니다.
시간순으로 정리했으니, 하나씩 확인하세요.

## 🔴 즉시 (1~3일)
- [ ] **사망진단서 발급** — 병원에서 최소 10통
- [ ] **사망신고** — 주민센터 (1개월 이내)
- [ ] **장례 진행** — 영수증 보관 (상속세 공제용)

## 🟠 1주 이내
- [ ] **가족관계증명서 발급** — 여러 통
- [ ] **안심상속 원스톱서비스 신청** — 주민센터/정부24
- [ ] **숨은 보험 조회** — 내보험찾아줌

## 🟡 1개월 이내
- [ ] **은행 계좌 정리** — 각 은행 방문 (무단 인출 금지!)
- [ ] **보험금 청구** — 각 보험사
- [ ] **통신 해지** — 구독 먼저 해지 후 통신 해지
- [ ] **카드 해지** — 자동결제 확인 후
- [ ] **구독 서비스 정리** — 앱, 카드 명세서 확인
- [ ] **유족연금 신청** — 국민연금공단

## 🔵 3개월 이내
- [ ] **상속포기/한정승인** — 가정법원 (⚠️ 기한 엄수!)

## 🟣 6개월 이내
- [ ] **상속세 신고** — 세무서 (3% 공제 혜택)
- [ ] **부동산 상속등기** — 등기소/법무사
- [ ] **자동차 명의이전** — 차량등록사업소

## ⚪ 이후
- [ ] 디지털 계정 정리 (네이버, 카카오, 구글 등)
- [ ] 운전면허 반납

## 📌 기한 놓치면?
| 절차 | 기한 | 불이익 |
|---|---|---|
| 사망신고 | 1개월 | 과태료 5만원 |
| 상속포기 | 3개월 | 빚 전액 상속 |
| 상속세 | 6개월 | 가산세 20~40% |
| 자동차 이전 | 6개월 | 과태료 최대 50만원 |
