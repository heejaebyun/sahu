## 피보팅 전략 (한국 시장 실패 시)

### 피보팅 트리거 조건

| 시점 | 지표 | 기준 | 판단 |
|------|------|------|------|
| 블로그 10편 발행 + 4주 | 주간 방문자 | < 50명 | 키워드 확장 피보팅 |
| 8주 경과 | 상담 신청 누적 | 0건 | 서비스 모델 피보팅 |
| 12주 경과 | 전문가 매칭 전환 | 0건 | 전면 피보팅 |

### 피보팅 순위

#### 1순위: "생애 행정 가이드"로 확장 (전환 비용 0원, 2주, 성공확률 70%)

사후뿐 아니라 이혼·창업·결혼·퇴직 등 모든 생애 전환점의 행정 체크리스트를 커버.

- **근거**: 사망 35만 + 결혼 19만 + 이혼 9만 + 창업 100만+ = 연간 잠재 사용자 10배 확대
- **전환 방법**: 기존 코드 그대로, `checklist-data.js`에 도메인별 데이터 추가 + 크롤링 키워드 교체
- **특히 유망**: "이혼 후 행정 절차" — 사후 행정과 구조 동일 (기한 체크리스트 + 전문가 매칭 + 감정적 난이도). 1일 내 전환 가능
- **리스크**: 도메인 전문성 희석. "사후" 브랜딩과 충돌 가능 → 상위 브랜드("행정길잡이" 등) 필요할 수 있음

#### 2순위: 상속 세무사 매칭 플랫폼 특화 (0원, 1주, 성공확률 50%)

가이드는 유입 도구로만 남기고, 세무사/법무사 매칭이 본업.

- **근거**: 상속세 신고 건당 수수료 100~500만원, 매칭 수수료 5~15% = 건당 10~50만원. TAM ~700억원
- **전환 방법**: 상담 폼을 매칭 전용으로 강화 + 세무사 3~5곳 직접 영업
- **리스크**: 트래픽 없이 세무사 제휴 설득 어려움. 닭-달걀 문제

#### 3순위: AI 에이전트 운영 키트 SaaS (월 5만원, 6주, 성공확률 30%)

현재 에이전트 시스템(CEO/CTO/마케팅/HR 에이전트 + 자동 회의 + 평가)을 "1인 창업자용 AI 회사 운영 도구"로 제품화.

- **근거**: 1인 창업자 한국 150만명. 월 3만원 구독 × 1,000명 = 월 3,000만원
- **전환 방법**: 에이전트 코드를 config 기반으로 범용화 + 웹 대시보드 구축
- **리스크**: B2C→B2B 전환이 큰 점프. 마케팅 채널 완전히 다름

### 피보팅 시 보존할 자산

| 자산 | 재활용 방법 |
|------|------------|
| 체크리스트 엔진 (D-day 계산) | 모든 기한 기반 행정에 범용 적용 |
| 지식IN 크롤링 파이프라인 | 키워드만 교체하면 어떤 도메인이든 질문 수집 |
| 블로그/브런치 자동 생성 | 프롬프트 템플릿만 교체 |
| 에이전트 회의 시스템 | 도메인 무관하게 작동 |
| Google Sheets 데이터 관리 | 구조 동일, 시트명만 변경 |
| SEO 인프라 (sitemap, robots, OG) | 그대로 유지 |
